## 🌟 About This Repository

This repo is my **JavaScript learning playground**.  
It contains assignments, challenges, and mini-projects I built to strengthen my core skills in **JavaScript, DOM manipulation, events, and APIs**.  
Each folder represents a different concept or practice project.

---

## 📂 Projects Included

- **API’s and Promise** – Working with APIs and JavaScript promises  
- **Calculator_JS** – A basic, fully functional calculator  
- **MyTodo** – A todo-list app with add/edit/delete functionality  
- **Quizapp** – Interactive quiz application  
- **digitalClock**, **stopWatch**, **weather** – Time and weather utility tools  
- **readmore** – Expand/collapse text component  
- **BakraEidAssignments** – Practice problems and small tasks
